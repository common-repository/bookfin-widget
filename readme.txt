== Description ==

*english version below*

Mit diesem Plugin k&ouml;nnen alle Blogger die neuesten eBooks von BookFin.de als Widget in ihrer Sidebar anzeigen. Man kann zwischen verschiedenen Optionen w&auml;hlen bzw. diese selbst setzen:

- Widget Titel

***English***

This plugin enables all bloggers to present the newest eBooks posted by BookFin.de as widget in your sidebar. You can choose and set various options:

- Widget title
- Amount of displayed vouchers
- and more...


== Installation ==

Die Installation ist sehr einfach. Wie bei allen Plugins muss es zun&auml;chst installiert werden:

In eurem Wordpress Adminbereich unter 'Plugins' auf 'Installieren' klicken, dann nach 'BookFin' suchen. Das gefundene Plugin mit einem Klick auf 'Jetzt installieren' einfach installieren lassen.

Dann musst du das Plugin aktivieren und folgendes durchf&uuml;hren:

1. Unter dem Men&uuml;punkt 'Design' auf 'Widgets' klicken und das Widget **BookFin eBook Widget** in eure Sidebar ziehen
1. Optionen beliebig anpassen.


***Alternative***

1. Upload `bookfin.widget.php` nach `/wp-content/plugins/` hochladen
1. Durch das 'Plugins' Menu in WordPress das Plugin **BookFin eBook Widget** aktivieren
1. Unter dem Men&uuml;punkt 'Design' auf 'Widgets' klicken und das Widget **bookfin.de Widget** in eure Sidebar ziehen
1. Optionen beliebig anpassen.
